# CARL — Central Arkansas Repair Leads

CARL is a Conway / Central Arkansas repair lead engine and local work board.

The first goal is simple: generate repair leads, especially drywall and small repair jobs, while building a local network of licensed contractors, handymen, skilled workers, and same-day labor help.

## Core user paths

### I Need Help
- Licensed Contractor
- Handyman / Small Repair
- Labor Help Today

### I’m Looking for Work
- Licensed Contractor
- Handyman / Skilled Worker
- Looking for Work Today

## First service vertical

Central Arkansas Drywall Repair

Focus:
- drywall holes
- patches
- texture repair
- ceiling repair
- rental damage
- move-out damage
- trade damage from electrical, alarm, HVAC, plumbing, and attic work

## MVP goal

Build a simple website with:
- landing page
- post-a-job form
- worker signup form
- contractor/handyman signup form
- drywall repair page
- admin dashboard mockup

No payments yet.
No full app login yet.
No Facebook API yet.
No over-engineering.

The first mission is leads.
